## Install
  - theme: `Options` → `Appearance` → `Themes` → `Manage` → type **Densy** → `Install and use`
    - or copy [manifest.json](https://github.com/eugenesvk/Obsidian-Densy/blob/main/manifest.json) [theme.css](https://github.com/eugenesvk/Obsidian-Densy/blob/main/theme.css) files to your vault's `.obsidian/themes/Densy` folder and then enable the theme in settings
